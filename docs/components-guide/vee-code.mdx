# Vee-Code Platform

Acceleration platform for building micro services and APIs, encapsulating the complexity involved in building these artifacts from a set of robust and customizable suggestions so that the user can focus on business rules to improve performance and decrease the learning curve of users.

O VKPR oferece dois tipos de Integração:

- [Dev-Portal](https://backstage.io/): Deploy, test and manage all your Specs and Services in one place. To see devportal commands, click [here](/docs/commands/devportal/install)

- [Mockserver](https://www.mock-server.com/): Make your Spec available with simulated data quickly and simply without the need for programming and deployment process. To see mockerserver commands, click [here](/docs/commands/mockerserver/install)
