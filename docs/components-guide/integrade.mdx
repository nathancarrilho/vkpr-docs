# Integrade Applications

Application integration connects two tools at a functional level, making the information between them generally take place in real-time.

VKPR offers two types of Integrations:

- [ArgoCD](https://argo-cd.readthedocs.io/en/stable/): Argo CD is a declarative GitOps continuous delivery tool for Kubernetes. To see argocd commands, click [here](/docs/commands/argocd/install)


- [PostgreSQL](https://www.postgresql.org/docs/): PostgreSQL is a related database manager. To see postgresql commands, click [here](/docs/commands/postgresql/install)
