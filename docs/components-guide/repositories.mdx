# Repositories

Repositories are places where developers can store their different code, they contain all the files of their project and the revision history of each file, being able to discuss and manage the project work within the repository.

VKPR offers operations on three types of Repositories:

- [Bitbucket](https://bitbucket.org/product/): Bitbucket Cloud is a Git-based code hosting and collaboration tool. To see bitbucket commands, click [here](/docs/commands/bitbucket/parameter/add)

- [Gitlab](https://docs.gitlab.com/): GitLab is the DevOps platform that empowers organizations to maximize the overall return of software development. To see gitlab commands, click [here](/docs/commands/gitlab/parameter/add)

- [Github](https://docs.github.com/pt): GitHub is a code hosting platform for version control and collaboration.  To see github commands, click [here](/docs/commands/github/parameter/add)
