# Security

Currently, companies tend to have sensitive data that must be stored and controlled.

VKPR offers certain types of Security, being them: 

- [Cert-Manager](https://cert-manager.io/docs/): Cert-Manager adds, renews, and validates certificates and certificate issuers as resource types in Kubernetes clusters. To see cert-manager commands, click [here](/docs/commands/cert-manager/install)

- [External-DNS](https://github.com/kubernetes-sigs/external-dns): ExternalDNS synchronizes exposed Kubernetes Services and Ingresses with DNS providers. To see external-dns commands, click [here](/docs/commands/external-dns/install)

- [Keycloak](https://github.com/keycloak/keycloak-documentation ou https://www.keycloak.org/documentation): Keycloak is an access and identity management solution. To see keycloak commands, click [here](/docs/commands/keycloak/install)

- [Vault](https://www.vaultproject.io/docs): HashiCorp Vault is an identity-based secrets and encryption management system. A secret is anything that you want to tightly control access to, such as API encryption keys, passwords, and certificates. To see vault commands, click [here](/docs/commands/vault/install)
