# Integration applications

Application integration connects two tools at a functional level, making the information between them generally take place in real-time.

VKPR offers four types of Integrations:

- [ArgoCD](https://argo-cd.readthedocs.io/en/stable/): ArgoCD is a declarative GitOps continuous delivery tool for Kubernetes. To see ArgoCD commands, click [here](/docs/commands/argocd/install)
- [Nexus](https://www.sonatype.com/products/nexus-repository): Nexus Repository OSS is an open source repository that supports many artifact formats, including Docker, Java, and npm. To see Nexus commands, click [here](/docs/commands/nexus/install)
- [PostgreSQL](https://www.postgresql.org/docs/): PostgreSQL is a related database manager. To see PostgreSQL commands, click [here](/docs/commands/postgresql/install)
- [Redis](https://redis.io/): Redis is an open source, in-memory data structure store, used as a database, cache, and message broker. To see Redis commands, click [here](/docs/commands/redis/install)
