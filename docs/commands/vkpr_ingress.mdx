# VKPR Ingress

## VKPR ingress install

### Description

Install nginx-ingress controller into cluster. Uses [ingress-nginx](https://artifacthub.io/packages/helm/ingress-nginx/ingress-nginx) Helm chart.

### Commands

Non-Interactive:

```bash
  vkpr ingress install
```

---

## VKPR ingress remove

### Description

Uninstall all the contet of nginx-ingress controller from the cluster.

### Commands

Non-interactive:

```bash
  vkpr ingress remove
```
