# VKPR cert-manager create wildcard

## Description

Create a wildcard certificate to the domain.

## Commands

Interactive inputs:

```bash
  vkpr cert-manager create wildcard [flags]
```

Non-interactive without setting values or using `VKPR Values`:

```bash
  vkpr cert-manager create wildcard --default
```

## Argument Reference

### Flags

```sh
  --domain  Set the application domain to create the wildcard
```
