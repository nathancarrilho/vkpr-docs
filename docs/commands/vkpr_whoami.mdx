# VKPR Whoami

## VKPR whoami install

### Description

Install whoami into cluster. Uses [whoami](https://artifacthub.io/packages/helm/cowboysysop/whoami) Helm chart.

### Commands

Interactive inputs:

```bash
  vkpr whoami install
```

Non-interactive:

```bash
  vkpr whoami install --domain="localhost" \
                      --secure=false
```

Non-interactive without setting values:

```bash
  vkpr whoami install --default
```

### Parameters

```bash
  --domain         Define the domain of whoami.      Default: localhost
  --secure         Define https on the whoami.       Default: false
  --default        Set all values with default.
```

### Values File Parameters

```jsx title="vkpr-values.yaml"
global:
  domain:               <String>
  secure:               <Boolean>
  whoami:
    ingressClassName:   <String>
    resources:          <Object>
    extraEnv:           <String>
```

## VKPR whoami remove

### Description

Uninstall all the content of whoami from the cluster.

### Commands

Non-interactive inputs:

```bash
  vkpr whoami remove
```
