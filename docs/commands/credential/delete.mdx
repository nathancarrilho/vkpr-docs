# VKPR credential delete

## Description

Remove the specified provider credentials.

## Comand

Interactive command:

```sh
 vkpr credential delete
```

#### Example:
```bash
$ vkpr credential delete
? type the provider name: (ex.:github, aws) [? for help] postgres
$ vkpr credential delete
Delete credential successful!
Check your credentials using vkpr credential list

```

### Flags

```bash
  vkpr credential delete [flags]
```

```sh
  --provider  Describe the provider whose credentials you want to delete in vkpr.
```
