# VKPR Init

## Description

Installs all binaries and their libraries used in VKPR.

Binaries are installed in `~/.vkpr/bin`, as follows:

- kubectl - v1.20.0
- helm - v3.9.0
- k3d - v5.2.2
- jq - v1.6
- yq - v4.16.1
- decK - v0.24.15
- aws-cli - latest
- okteto-cli - latest

:::note Attention!
This formula is used only when you want to contribute with VKPR or update the VKPR binaries.
:::

## Commands

Non-interactive:

```bash
  vkpr init
```
