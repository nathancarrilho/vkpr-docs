# VKPR whoami install

## Description

Install whoami into cluster. For more information about whoami, click [here](https://github.com/traefik/whoami).

## Commands

Interactive inputs:

```bash
  vkpr whoami install [flags]
```

Non-interactive without setting values or using `VKPR Values`:

```bash
  vkpr whoami install --default
```

### Parameters

```txt
  --default           Set all values with default.
  --domain            Define the domain used by the whoami.   Default: localhost
  --secure            Specifies if the application will have HTTPS.    Default: false
```

### Values File Parameters

```jsx title="vkpr.yaml"
global:
  domain:               <String>
  secure:               <Boolean>
  whoami:
    namespace:          <String>
    ingressClassName:   <String>
    helmArgs:           <Object>
```
