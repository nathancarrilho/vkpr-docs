# VKPR Prometheus-Stack

## VKPR prometheus-stack install

### Description

Install Prometheus-Stack into cluster. Uses [Prometheus-Stack](https://artifacthub.io/packages/helm/prometheus-community/kube-prometheus-stack) Helm chart.

### Commands

Interactive inputs:

```bash
  vkpr prometheus-stack install
```

Non-interactive:

```bash
  vkpr prometheus-stack install --domain="localhost" \
                                --secure=false \
                                --grafana_password="vkpr123" \
                                --alertmanager="false"
```

Non-interactive without setting values:

```bash
  vkpr prometheus-stack install --default
```

### Parameters

```bash
  --domain            Define the domain of grafana.             Default: localhost
  --secure            Define https on the grafana.              Default: false
  --grafana_password  Define RBAC Admin password grafana.       Default: vkpr123
  --alertmanager      Define is installs alertmanager.          Default: false
  --default           Set all values with default.
```

### Values File Parameters

```jsx title="vkpr-values.yaml"
global:
  domain:               <String>
  secure:               <Boolean>
  prometheus-stack:
    ingressClassName:   <String>
    alertmanager:       <String>
```

## VKPR prometheus-stack remove

### Description

Uninstall all the content of Prometheus-Stack from the cluster.

### Commands

Non-interactive inputs:

```bash
  vkpr prometheus-stack remove
```
