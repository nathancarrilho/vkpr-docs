# VKPR okteto init

## Description

Create or associate to a cluster in Okteto cloud. For more information about Okteto, click [here](https://www.okteto.com/).

## Commands

Interactive inputs:

```bash
  vkpr okteto init [flags]
```

Non-interactive without setting values or using `VKPR Values`:

```bash
  vkpr okteto init --default
```
