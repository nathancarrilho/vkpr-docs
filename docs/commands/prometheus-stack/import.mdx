# VKPR prometheus-stack import

## Description

Import a new Dashboard local file to Grafana

## Commands

Interactive inputs:

```bash
  vkpr prometheus-stack import [flags]
```

## Argument Reference

### Flags

```sh
  --dashboard_path     Specifies the path from dashboard file.
```
