# VKPR infra stop

## Description

Destroy the k3d local cluster.

## Commands

Interactive inputs:

```bash
  vkpr infra stop [flags]
```

Non-interactive without setting values or using `VKPR Values`:

```bash
  vkpr infra stop --default
```

## Argument Reference

### Flags

```sh
  --default=false: Use to automatically fill inputs with default values.
  --delete_registry=false: Specifies if you want to delete the builtin registries
```
