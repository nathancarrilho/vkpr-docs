# VKPR infra stop

## Description

Destroy the k3d local cluster with the registries containers.

## Commands

Interactive inputs:

```bash
  vkpr infra stop [flags]
```

## Argument Reference

### Flags

```sh
  --default          Use to automatically fill inputs with default values.
  --delete_registry  Specifies if you want to delete the builtin registries
```
