# VKPR infra up

## Description

Create a k3d cluster using [VKPR infra start](/docs/commands/infra/vkpr_infra_start) defaults.

## Commands

Non-Interactive:

```bash
  vkpr infra up
```

---

### See Also

- [VKPR infra start](/docs/commands/infra/vkpr_infra_start) - Create a configurable local cluster with k3d.
- [VKPR infra down](/docs/commands/infra/vkpr_infra_down) - Delete the local cluster created.
