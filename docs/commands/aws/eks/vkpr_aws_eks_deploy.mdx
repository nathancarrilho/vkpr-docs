# VKPR aws eks deploy

### Description

Deploy the configuration generated by the Gitops pipeline to AWS.

### Commands

Non-interactive:

```bash
  vkpr aws eks deploy [flags]
```

### Parameters

```txt
  --gitlab_token                    Specifies your Gitlab Access-Token.
  --gitlab_username                 Specifies your Gitlab Username.
```
