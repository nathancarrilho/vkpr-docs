# VKPR aws rds destroy

## Description

Fork and setup the [GitOps Repo](https://gitlab.com/vkpr/aws-eks) in your Gitlab, using Terraform Cloud as Backend to save the Terraform state.

## Commands

Interactive inputs:

```bash
  vkpr aws rds destroy [flags]
```

Non-interactive without setting values or using `VKPR Values`:

```bash
  vkpr aws rds destroy --default
```

## Argument Reference

### Flags

```sh
  --rds_instance_name='rds-sample': Specifies the RDS instance name to destroy.
```
