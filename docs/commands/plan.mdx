# VKPR plan

## Description

VKPR Plan is a simpler method of validating the installation and possible updates of applications that you want to install with VKPR. With a yaml file informing the values ​​that the applications will use, it will generate a preview of all the Kubernetes artifacts that will be generated and will display in an interactive terminal for visualization.

## Commands

Interactive inputs:

```bash
  vkpr plan [flags]
```

Non-interactive:

```bash
  vkpr plan --path_to_file="/path/to/vkpr.yaml"
```
