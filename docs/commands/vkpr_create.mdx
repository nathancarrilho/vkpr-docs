# VKPR Create

## VKPR create formula

### Description

Create a new formula to use in VKPR.

:::note Attention!
  This formula is used only when you want to contribute with VKPR
:::

### Commands

Interactive inputs:

```bash
  vkpr create formula
```

Non-interactive:

```bash
  vkpr create formula --vkpr_formula="name-formula"
```
