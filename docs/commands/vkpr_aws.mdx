# VKPR AWS

## VKPR aws eks up

### Description

Fork and setup the [GitOps Repo](https://gitlab.com/vkpr/aws-eks) in your Gitlab.

### Commands

Non-Interactive:

```bash
  vkpr aws eks up
```

---

## VKPR aws eks deploy

### Description

Deploy the configuration generated by the Gitops pipeline to AWS.

### Commands

Non-interactive:

```bash
  vkpr aws eks deploy
```

---

## VKPR aws eks destroy

### Description

Destroy the k3d cluster.

### Commands

Interactive inputs:

```bash
  vkpr aws eks destroy
```

Non-interactive:

```bash
  vkpr aws eks down --delete_registry=false
```

Non-interactive without setting values:

```bash
  vkpr aws eks down --default
```

### Parameters

```bash
  --delete_registry   Delete Registry and Mirror Registry.   Default: false
  --default           Set all values with default.
```

:::info AWS
  For more information about AWS EKS, [click here](https://aws.amazon.com/pt/eks/)
:::
