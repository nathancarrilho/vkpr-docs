# VKPR kong deck sync

## Description

Import content into kong workspace.

## Commands

Interactive inputs:

```bash
  vkpr kong deck sync [flags]
```

## Argument Reference

### Flags

```sh
  --default           Use to automatically fill inputs with default values
  --kong_addr         Specifies the Kong Admin Adress
  --kong_admin_token  Specifies the Kong Admin Token
  --kong_workspace    Specifies the Kong Workspace
  --kong_yaml_path    Type the path to the folder with your kong.yaml file
```
