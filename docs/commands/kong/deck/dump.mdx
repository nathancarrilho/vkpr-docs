# VKPR kong deck dump

## Description

Export kong workspace content.

## Commands

Interactive inputs:

```bash
  vkpr kong deck dump [flags]
```

## Argument Reference

### Flags

```sh
  --default           Use to automatically fill inputs with default values
  --kong_addr         Specifies the Kong Admin Adress
  --kong_admin_token  Specifies the Kong Admin Token
  --kong_workspace    Specifies the Kong's Workspace
```
