# VKPR keycloak import realm

## Description

Import realm from the specifed file in Keycloak database.

## Commands

Interactive inputs:

```bash
  vkpr keycloak import realm [flags]
```

## Argument Reference

### Flags

```sh
  --keycloak_password='': Specifies the Keycloak Super admin password.
  --keycloak_username='': Specifies the Keycloak Super admin username.
  --realm_path='': Specifies the path to file.
```
