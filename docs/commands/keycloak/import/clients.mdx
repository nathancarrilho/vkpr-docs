# VKPR keycloak import clients

## Description

Import clients from a specifed file in Keycloak database.

## Commands

Interactive inputs:

```bash
  vkpr keycloak import clients [flags]
```

## Argument Reference

### Flags

```sh
  --keycloak_password='': Specifies the Keycloak Super admin password.
  --keycloak_username='': Specifies the Keycloak Super admin username.
  --realm_name='': Specifies the realm name to be imported.
  --realm_path='': Specifies the path to file.
```
