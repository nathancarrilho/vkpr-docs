# VKPR keycloak realm idp

## Description

Add a Identity Provider in the realm.

## Commands

Interactive inputs:

```bash
  vkpr keycloak realm idp [flags]
```

## Argument Reference

### Flags

```sh
  --realm_name         Specifies the realm name.
  --keycloak_password  Specifies the Keycloak Super admin password.
  --keycloak_username  Specifies the Keycloak Super admin username.
  --provider_name      Specifies which provider will be configured.
```
