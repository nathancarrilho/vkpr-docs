# VKPR keycloak export realm

## Description

Export a realm from the Keycloak database.

## Commands

Interactive inputs:

```bash
  vkpr keycloak export realm [flags]
```

### Parameters

```txt
  --keycloak_password       Specifies the Keycloak Super admin password
  --realm_name              Specifies the realm name to be exported
```
