# VKPR keycloak export users

## Description

Export all users from the realm specified in Keycloak database.

## Commands

Interactive inputs:

```bash
  vkpr keycloak export users [flags]
```

### Parameters

```txt
  --keycloak_password       Specifies the Keycloak Super admin password
  --realm_name              Specifies the realm name to be exported
```
