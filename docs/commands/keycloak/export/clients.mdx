# VKPR keycloak export clients

## Description

Export all clients from the realm specified in Keycloak database.

## Commands

Interactive inputs:

```bash
  vkpr keycloak export clients [flags]
```

## Argument Reference

### Flags

```sh
  --keycloak_password='': Specifies the Keycloak Super admin password.
  --keycloak_username='': Specifies the Keycloak Super admin username.
  --realm_name='': Specifies the realm name to be imported.
```
