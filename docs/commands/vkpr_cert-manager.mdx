# VKPR Cert-manager

## VKPR cert-manager install do

### Description

Install cert-manager to Digital Ocean into cluster. Uses [cert-manager](https://artifacthub.io/packages/helm/bitnami/cert-manager) Helm chart.

### Commands

Interactive inputs:

```bash
  vkpr cert-manager install do
```

Non-interactive:

```bash
  vkpr cert-manager install do --email="email@email.com" \
                               --issuer="letsencrypt-staging"
```

Non-interactive without setting values:

```bash
  vkpr cert-manager install do --default
```

### Parameters

```bash
  --issuer        Define the ClusterIssuer                  Default: letsencrypt-staging
  --email         Define the email to generate certificates
  --default       Set all values with default.
```

### Setting Digital Ocean credentials

#### Digital Ocean
```bash
  rit set credential --provider="digitalocean" --fields="token" --values="your-api-token"
```

### Values File Parameters

```jsx title="vkpr-values.yaml"
global:
  cert-manager:
    email:         <String>
```

## VKPR cert-manager install custom-acme

### Description

Install cert-manager using custom-acme into cluster.

### Commands

Interactive inputs:

```bash
  vkpr cert-manager install custom-acme
```

Non-interactive:

```bash
  vkpr cert-manager install custom-acme --email="email@email.com"
```

### Parameters

```bash
  --email         Define the email to generate certificates
```

## VKPR cert-manager remove

### Description

Uninstall all the content of cert-manager from the cluster.

### Commands

Non-interactive inputs:

```bash
  vkpr cert-manager remove
```
