# VKPR Loki

## VKPR loki install

### Description

Install loki into cluster. Uses [loki](https://artifacthub.io/packages/helm/grafana/loki-stack) Helm chart.

### Commands

Interactive inputs:

```bash
  vkpr loki install
```

Non-interactive:

```bash
  vkpr loki install --domain="localhost"
```

Non-interactive without setting values:

```bash
  vkpr loki install --default
```

### Parameters

```bash
  --domain         Define the domain of loki.      Default: localhost
  --default        Set all values with default.
```

### Values File Parameters

```jsx title="vkpr-values.yaml"
global:
  domain:               <String>
```
---

## VKPR loki remove

### Description

Uninstall all the content of loki from the cluster.

### Commands

Non-interactive inputs:

```bash
  vkpr loki remove
```
