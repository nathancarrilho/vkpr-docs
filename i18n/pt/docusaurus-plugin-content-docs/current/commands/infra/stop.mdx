# VKPR infra stop

## Descrição

Destroi o cluster K3D juntamente com seus registries.

## Comando

Comando interativo:

```bash
  vkpr infra stop [flags]
```

## Referência de valores

### Flags

```sh
  --default          Use para preencher automaticamente as entradas com valores padrões
  --delete_registry  Específica se deseja destruir os registries criados com docker
```
