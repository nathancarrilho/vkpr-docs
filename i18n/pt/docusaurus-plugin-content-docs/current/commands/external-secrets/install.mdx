# VKPR external-secrets install

## Descrição

Instala o external-secrets no cluster. Para mais informações sobre external-secrets, clique [aqui](https://external-secrets.io/v0.6.0-rc1/).

## Comando

```bash
  vkpr external-secrets install
```
