# VKPR kong deck sync

## Descrição

Importa conteúdo dentro do Kong Workspace.

## Comando

Comando interativo:

```bash
  vkpr kong deck sync [flags]
```

## Referência de valores

### Flags

```sh
  --default           Use para preencher automaticamente as entradas com valores padrões
  --kong_addr         Específica a URL do Kong admin
  --kong_admin_token  Específica a senha do admin
  --kong_workspace    Específica o Workspace do Kong a ser exportado
  --kong_yaml_path    Informa o caminho do arquivo de conf do Kong
```
