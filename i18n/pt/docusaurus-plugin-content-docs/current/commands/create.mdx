# VKPR Create Formula

## Descrição

Crie uma nova fórmula para usar no VKPR.

:::warning Cuidado!
Como o VKPR foi construído em cima do Ritchie, ele precisa das dependências do Ritchie para criar as estruturas necessárias a execução da fórmula. Para isso você precisará ter executado previamente o comando `rit init`.
:::

:::note Atenção!
Essa fórmula é usada apenas quando você deseja contribuir com o VKPR.
:::

## Comando

Comando interativo:

```bash
  vkpr create formula [flags]
```

Comando não interativo:

```bash
  vkpr create formula --vkpr_formula="name-formula"
```
