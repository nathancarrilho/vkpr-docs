# VKPR postgresql createdb

## Descrição

Crie um novo database no PostgreSQL com um owner.

## Comando

Comando interativo:

```bash
  vkpr postgresql createdb [flags]
```

## Referência de valores

### Flags

```txt
  --default     Use para preencher automaticamente as entradas com valores padrões
  --dbname      Específica o nome do novo database
  --dbuser      Específica o username do database owner
  --dbpassword  Específica a senha do owner do database
```
