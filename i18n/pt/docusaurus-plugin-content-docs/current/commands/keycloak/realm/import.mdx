# VKPR keycloak realm import

## Descrição

Importa um realm com base em um arquivo próprio para o Keycloak.

## Comando

Comando interativo:

```bash
  vkpr keycloak realm import [flags]
```

## Referência de valores

### Flags

```sh
  --realm_path         Define o caminho do arquivo do Realm
  --keycloak_password  Específica o username do admin
  --keycloak_username  Específica a senha do admin
```
