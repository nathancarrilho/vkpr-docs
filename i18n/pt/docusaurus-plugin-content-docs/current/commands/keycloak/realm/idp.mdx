# VKPR keycloak realm idp

## Description

Adiciona Identity Provider a um realm.

## Comando

Comando interativo:

```bash
  vkpr keycloak realm idp [flags]
```

## Referência de valores

### Flags

```sh
  --realm_name         Nome do Realm a adicionar o IdP
  --keycloak_password  Específica o username do admin
  --keycloak_username  Específica a senha do admin
  --provider_name      Define qual IdP será adicionado
```
