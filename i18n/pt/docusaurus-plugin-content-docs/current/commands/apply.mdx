# VKPR Apply

## Descrição

VKPR é um método mais simples de instalar todas as aplicações que você deseja com o VKPR. Com um arquivo yaml informando os valores que as aplicações irão utilizar, ele executará todas as fórmulas necessárias para criar o cenário que foi especificado no arquivo yaml.

## Comando

Comando interativo:

```bash
  vkpr apply [flags]
```

Comando não interativo:

```bash
  vkpr apply --path_to_file="/path/to/vkpr.yaml"
```
