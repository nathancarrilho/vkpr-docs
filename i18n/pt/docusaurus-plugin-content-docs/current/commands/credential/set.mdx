# VKPR credential set

## Descrição

Crie credenciais de um provider para serem utilizadas pelo vkpr.

## Comando

Comando interativo:

```sh
 vkpr credential set
```
#### Exemplo:

```bash
$ vkpr credential set
Formula was successfully built!
? Select your provider: postgres
? Define your field name: (ex.:token, secretAccessKey) password123
? Set value for previous field: (ex.:password123, secretAccessKey) vkpr123
```

### Flags

```bash
  vkpr credential set [flags]
```

```sh
  --provider  Descreva o provider o qual deseja criar as credenciais no vkpr.
  --fields    Descreva o campo ou os campos nos quais as credenciais serão adicionada no provider. 
  --values    Descreva o valor ou os valores que serão atribuidos aos campos do provider adicionado.
```
