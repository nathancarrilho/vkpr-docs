# VKPR okteto init

## Descrição

Cria ou associa a um cluster do Okteto Cloud. Para mais informações sobre o Okteto, clique [aqui](https://www.okteto.com/).

## Comando

Comando interativo:

```bash
  vkpr okteto init
```
