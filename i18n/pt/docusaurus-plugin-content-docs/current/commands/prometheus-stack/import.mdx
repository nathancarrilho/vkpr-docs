# VKPR prometheus-stack import

## Descrição

Importa um Dashboard por um arquivo para o Grafana.

## Comando

Comando interativo:

```bash
  vkpr prometheus-stack import [flags]
```

## Referência de valores

### Flags

```sh
  --dashboard_path   Específica o caminho do arquivo do dashboard
```
