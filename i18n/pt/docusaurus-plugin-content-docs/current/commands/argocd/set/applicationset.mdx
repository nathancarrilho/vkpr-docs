# VKPR argocd set applicationset

## Descrição

Gera um manifesto do Addon Applicationset que irá aplicar todos os manifestos da aplicação do repositório apontado.

## Comando

Comando interativo:

```bash
  vkpr argocd set applicationset [flags]
```

## Referência de valores

### Flags

```sh
  --repo_url  URL do repositório Git para sincronizar com o ArgoCD
```
