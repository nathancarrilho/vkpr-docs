# VKPR aws eks kubeconfig

## Descrição

Faz Download do Kubeconfig para acesso ao EKS Cluster.

:::info
Para utiliza-lo, execute o comando:<br/>
`export KUBECONFIG=$HOME/.vkpr/kubeconfig/aws-eks/kubeconfig_<nome-do-cluster>_<nome-do-usuário>`
:::

## Comando

Comando não interativo:

```bash
  vkpr aws eks kubeconfig [flags]
```
