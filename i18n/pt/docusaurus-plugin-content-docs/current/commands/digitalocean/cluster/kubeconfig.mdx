# VKPR digitalocean cluster kubeconfig

## Descrição

Faz Download do Kubeconfig para acesso ao Cluster.

:::info
Para utiliza-lo, execute o comando:<br/>
`export KUBECONFIG=$HOME/.vkpr/kubeconfig/k8s-digitalocean/kubeconfig_<nome-do-cluster>_<nome-do-usuário>`
:::

## Comando

Comando não interativo:

```bash
  vkpr aws digitalocean kubeconfig [flags]
```
