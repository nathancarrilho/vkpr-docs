# VKPR cert-manager create wildcard

## Descrição

Cria um certificicado wildcard para o domínio específicado.

## Comando

Comando interativo:

```bash
  vkpr cert-manager create wildcard [flags]
```

Comando não interativo ou utilizando o `VKPR Values`:

```bash
  vkpr cert-manager create wildcard --default
```

## Referência de valores

### Flags

```sh
  --domain  Descreva o domínio o qual deseja criar o certificado wildcard
```
