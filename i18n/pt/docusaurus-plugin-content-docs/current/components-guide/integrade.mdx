# Aplicações Integradas

A integração de aplicações conecta duas ferramentas em um nível funcional, fazendo com que a informação entre elas ocorram geralmente em tempo-real.

O VKPR oferece dois tipos de Integração:

- [ArgoCD](https://argo-cd.readthedocs.io/en/stable/): Argo CD é uma ferramenta declarativa de entrega contínua GitOps para Kubernetes. Para visualizar os comandos do argocd, clique [aqui](/docs/commands/argocd/install)


- [PostgreSQL](https://www.postgresql.org/docs/): PostgreSQL é um gerenciador de banco de dados relacionados. Para visualizar os comandos do postgresql, clique [aqui](/docs/commands/postgresql/install)
