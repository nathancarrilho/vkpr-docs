# Repositórios

Os repositórios são lugares onde os desenvolvedores podem armazenar seus diferentes códigos, contém todos os arquivos do seu projeto e o histórico de revisão de cada arquivo podendo discutir e gerenciar o trabalho do projeto dentro do repositório.

O VKPR oferece operações em três tipos de Repositórios:

- [Bitbucket](https://bitbucket.org/product/): O Bitbucket Cloud é uma ferramenta de hospedagem e colaboração de código baseada em Git. Para visualizar os comandos do BitBucket, clique [aqui](/docs/commands/bitbucket/parameter/add).
- [Gitlab](https://docs.gitlab.com/): O GitLab é a plataforma DevOps que capacita as organizações a maximizar o retorno geral do desenvolvimento de software. Para visualizar os comandos do Gitlab, clique [aqui](/docs/commands/gitlab/parameter/add).
- [Github](https://docs.github.com/pt): O GitHub é uma plataforma de hospedagem de código para controle de versão e colaboração. Para visualizar os comandos do Github, clique [aqui](/docs/commands/github/parameter/add).
