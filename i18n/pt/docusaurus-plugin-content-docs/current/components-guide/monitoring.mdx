# Monitoramento e Registro

Ao dimensionar e aprimorar aplicativos com o K8S, é necessário em algum momento verificar se as aplicações estão funcionando conforme o esperado. Ao monitorar suas aplicações, você pode ver que sua infraestrutura está subutilizando recursos que poderiam ser salvos.

O VKPR disponibiliza a integração de determinadas ferramentas de monitoramento para uso em seu cluster, sendo elas:

- [Grafana](https://grafana.com/grafana/): Dashboard para visualizar todos os dados das suas aplicações. Para visualizar os comandos do Grafana, clique [aqui](/docs/commands/prometheus-stack/install)
- [Loki](https://grafana.com/oss/loki/): Usado para monitorar logs das aplicações. Para visualizar os comandos do Loki, clique [aqui](/docs/commands/loki/install)
- [Jaeger](https://github.com/jaegertracing/documentation): Usado para monitorar e solucionar problemas de sistemas distribuídos baseados em microsserviços. Para visualizar os comandos do Jaeger, clique [aqui](/docs/commands/jaeger/install)
- [Prometheus](https://prometheus.io/): Usado para monitorar e alertar aplicações. Para visualizar os comandos do Prometheus, clique [aqui](/docs/commands/prometheus-stack/install)
