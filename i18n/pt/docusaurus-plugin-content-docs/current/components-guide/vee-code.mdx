# VeeCode Platform

Platforma de aceleração para construção de micro serviços e APIs, encapsulando a complexidade que envolve a construção desses artefatos a partir de um conjunto de sugestões robustas e customizáveis para que usuário possa se concentrar nas regras de negócios visando melhoria de desempenho e diminuição da curva de aprendizado dos usuárioos.

O VeeCode Plataform utiliza algumas instalações próprias da VKPR, sendo elas:

- [Dev-Portal](https://backstage.io/): Disponibilize, teste e gerencie todas as suas Specs e Services em um só lugar. Para visualizar os comandos do DevPortal, clique [aqui](/docs/commands/devportal/install)
- [Mockserver](https://www.mock-server.com/): Disponibilize suas Spec com dados simulados de forma rápida e simples sem a necessidade de programação e processo de deploy. Para visualizar os comandos do mockserver, clique [aqui](/docs/commands/mockserver/install)
